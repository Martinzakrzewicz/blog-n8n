¿Qué es n8n?

n8n es una herramienta de automatización de código abierto que permite crear flujos de trabajo para conectar múltiples aplicaciones entre sí. Es ideal para automatizar procesos sin necesidad de escribir mucho código.

Casos de Uso Comunes

  • Enviar correos automáticos cuando recibes un formulario.
  • Publicar contenido en redes sociales de forma programada.
  • Integrar Google Sheets con otros servicios.
  • Crear alertas cuando ciertos datos cambian.

Tutorial Básico: Automatiza un Correo desde un Formulario

  1. Instala n8n en tu PC o usa la versión en la nube.
  2. Crea un nuevo flujo de trabajo.
  3. Añade un nodo "Webhook" para recibir datos de un formulario.
  4. Conecta un nodo "Email" para enviar los datos recibidos.
  5. Activa el flujo y pruébalo.
