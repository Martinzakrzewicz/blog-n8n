¿Qué es n8n?

n8n (pronunciado "n-eight-n") es una herramienta de workflow automation de código abierto que permite conectar servicios web, aplicaciones y bases de datos para crear flujos automatizados. A diferencia de plataformas como Zapier o Make, n8n se puede auto hospedar y permite una lógica más avanzada gracias a su editor visual y capacidad de ejecutar código JavaScript en los nodos.

Ideal para desarrolladores, empresas y entusiastas del No-Code/Low-Code, n8n ofrece una solución escalable, flexible y económica.

Ventajas de usar n8n

  • Open Source: completamente libre y modificable.
  • Sin límites de tareas: a diferencia de otras plataformas, puedes ejecutar tantos flujos como quieras.
  • Integración con +350 servicios: desde Gmail hasta bases de datos SQL.
  • Auto-hospedaje: puedes tener el control total alojando n8n en tu propio servidor o en servicios como Heroku, DigitalOcean o Docker.
  • Flujos condicionales, bucles y funciones personalizadas para lógica avanzada.

Casos Reales de Automatización

1. Automatización de CRM

Un equipo de ventas conecta su formulario de contacto (Typeform) con HubSpot, Google Sheets y Slack. Cuando un cliente llena el formulario:

  • Los datos se almacenan en Sheets.
  • Se crea un contacto en HubSpot.
  • Se notifica automáticamente al canal de ventas en Slack.

2. Generación de Reportes Financieros

Conectando bases de datos MySQL, n8n extrae datos financieros, los transforma y envía por correo un PDF con el informe mensual.

3. Publicación automática en redes sociales

Desde una hoja de Google, n8n publica contenido en LinkedIn y Twitter según fechas programadas, utilizando nodos de delay y verificación de horario.

Tutorial Básico: Automatización de Correos desde un Formulario

  1. Ve a n8n cloud o instala n8n localmente con Docker.
  2. Crea un nuevo flujo de trabajo.
  3. Añade un nodo Webhook y copia la URL que se genera.
  4. Conéctalo a un nodo Set para estructurar los datos.
  5. Conéctalo a un nodo Gmail o Email para enviar los datos.
  6. Haz clic en "Activate" y prueba enviando datos desde un formulario HTML.
